VeinTrack

Privacy Policy

Effective date: To be set when published.

VeinTrack is designed to work without a Duck Code backend account. The app is focused on private, on-device tracking. Data is stored on the user's device unless the user chooses to export, share, back up, email, or attach it elsewhere.

Data The App May Handle

Depending on how the user configures and uses the app, VeinTrack may handle:

  • user-entered health context, notes, and event details,
  • pain and swelling scores,
  • leg elevation and rest records,
  • compression use and compression prescription details,
  • activity session records, steps, walking/running distance, and workout-related context,
  • sleep and weight data from Apple Health when permission is granted,
  • Motion activity data when permission is granted,
  • photos attached by the user,
  • consultation details and attachments added by the user,
  • travel and Flight Mode records,
  • generated reports, PDFs, HTML summaries, CSV exports, and backups,
  • optional support purchase status provided by Apple's StoreKit systems.

Apple Health And Motion

VeinTrack only accesses Apple Health or Motion data after the user grants permission. Apple Health permissions can be managed in the Health app or iOS Settings. Motion permissions can be managed in iOS Settings.

Health and Motion data is used to support app features such as activity summaries, rest reminders, reports, and insights. VeinTrack does not upload Apple Health or Motion data to a Duck Code backend.

Photos And Files

Users may attach photos and files to events or reports. These remain part of the user's local app data unless the user chooses to export, share, email, back up, or save them elsewhere.

Notifications, Widgets, And Live Activities

VeinTrack may use notifications, widgets, and Live Activities to show reminders and activity/rest session status. These features are controlled by iOS permissions and user settings.

Optional Support

VeinTrack may offer optional support purchases through Apple StoreKit. These purchases are voluntary and do not unlock features. Purchase processing is handled by Apple.

Feedback Email

If the user sends feedback by email, the email is handled by the user's chosen mail provider and received at duck_code_contact@proton.me. Users should avoid sending sensitive information unless they choose to do so.

Data Sale And Tracking

Duck Code does not sell VeinTrack user data. VeinTrack does not require a Duck Code account and does not upload personal health data to a Duck Code backend.

Deleting Data

Deleting the app deletes local app data from the device, except for any data the user previously exported, shared, backed up, emailed, saved to Files, or stored in external services.

Contact

Questions or privacy requests can be sent to duck_code_contact@proton.me.
